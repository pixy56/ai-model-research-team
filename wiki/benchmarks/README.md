# AI Benchmarks

Key benchmarks for evaluating AI models.

## Core Benchmarks

### MMLU (Massive Multitask Language Understanding)
- **Description:** 57 tasks covering STEM, humanities, social sciences
- **Metric:** Accuracy (%)
- **Source:** [Papers with Code](https://paperswithcode.com/sota/multi-task-language-understanding-on-mmlu)

### HumanEval
- **Description:** Code generation benchmark (164 problems)
- **Metric:** Pass@k
- **Source:** [OpenAI](https://github.com/openai/human-eval)

### GPQA (Graduate-Level Google-Proof Q&A)
- **Description:** Graduate-level science questions
- **Metric:** Accuracy (%)
- **Source:** [NYU](https://github.com/idavidrein/gpqa)

### MATH
- **Description:** Competition mathematics problems
- **Metric:** Accuracy (%)
- **Source:** [Hugging Face](https://huggingface.co/datasets/hendrycks/competition_math)

### SWE-bench
- **Description:** Real-world software engineering tasks
- **Metric:** Resolution rate (%)
- **Source:** [Princeton](https://www.swebench.com/)

## Multimodal Benchmarks

### MMMU (Massive Multi-discipline Multimodal Understanding)
- **Description:** College-level multimodal problems
- **Metric:** Accuracy (%)

### MMBench
- **Description:** Comprehensive multimodal evaluation
- **Metric:** Accuracy (%)

## Reasoning Benchmarks

### ARC (AI2 Reasoning Challenge)
- **Description:** Grade-school science questions
- **Metric:** Accuracy (%)

### HellaSwag
- **Description:** Commonsense reasoning
- **Metric:** Accuracy (%)
