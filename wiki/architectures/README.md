# AI Model Architectures

Classification of architectural innovations in latest AI models.

## Architecture Types

### Dense Transformers
Standard transformer architecture with full attention.

### Mixture of Experts (MoE)
Sparse activation patterns for efficiency.

### State Space Models (SSM)
Linear attention alternatives (Mamba, etc.).

### Multimodal Architectures
Vision-language models, audio-text integration.

### Reasoning Architectures
Chain-of-thought, tree-of-thought, test-time compute.
