# AI Model Research Team

SAFe Agentic Research Team focused on tracking and analyzing the latest AI models.

## Quick Start

1. Review the [SAFe Research Team Plan](../.hermes/plans/2025-04-19_SAFe_Research_Team_Setup.md)
2. Check the [PI Planning Document](docs/PI-1-Planning.md)
3. View current [Iteration Backlog](https://github.com/projects)

## Team Structure

- **Product Owner:** Research Lead
- **Scrum Master:** Process Facilitator  
- **Dev Team:** Your Development Team
- **AI Agents:**
  - Literature Review Agent
  - Data Analysis Agent
  - Writing Agent
  - Integration Agent

## Repository Structure

```
├── agents/           # Agent configurations
├── data/            # Research data
├── docs/            # Documentation
├── scripts/         # Automation scripts
├── notebooks/       # Analysis notebooks
├── src/             # Source code (Dev Team)
├── tests/           # Test suite
└── wiki/            # llm-wiki knowledge base
```

## Ceremonies

- **PI Planning:** Every 10 weeks
- **Iteration:** 2 weeks
- **Standup:** Daily (async)
- **Review/Retro:** End of iteration
